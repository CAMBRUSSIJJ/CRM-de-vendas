# CRM v62 — Garimpo de Leads e Personalização Visual

## Ajustes feitos

1. Criação da página oficial **Garimpo de Leads**.
2. Inclusão do Garimpo dentro da área **Leads** da sidebar compacta.
3. Página de garimpo com:
   - critérios de segmento, cidade, palavras-chave e ticket;
   - sugestões locais com score comercial;
   - links para Google Maps e Google Search;
   - importação por lista colada;
   - seleção em massa;
   - envio dos leads selecionados para o CRM;
   - exportação CSV.
4. Painel de personalização visual dentro de Configurações.
5. Correção do modo claro/escuro com controle próprio v62.
6. Personalização de cores:
   - cor principal;
   - cor de destaque;
   - fundo claro;
   - cor do texto;
   - paletas prontas.

## Observação importante

O CRM publicado em GitHub Pages não consegue raspar automaticamente dados do Google Maps, porque isso exige API/backend e regras de uso específicas. A nova página funciona com pesquisa assistida, importação manual e estrutura pronta para conectar uma API própria no futuro.
